Close Your Tabs

Tab Management Extension and Web Portal

A web portal which incorperates a chrome extension focused on showing each tab's inactive time.

Using Google Oauth2 for our login, we created an interactive user experience which tracks the amount of time since each tab's last activity. The target audiences are students, developers, and those who refuse to close browser tabs.

Project details
  • Major Frameworks: React, Node.js, MySQL
  • Website : closeyourtabs.com
